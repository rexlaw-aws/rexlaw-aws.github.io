    Windows Server 版本和授權¶

    Windows Server 版本¶

    Amazon EKS 優化的 Windows AMI 基於 Windows Server 2019 和 2022 Datacenter 版本的長期服務頻道 (LTSC)。Datacenter 版本沒有對工作節點上運行的容器數量的限制。更多信息請參考: https://docs.microsoft.com/en-us/virtualization/windowscontainers/about/faq

    長期服務頻道 (LTSC)¶

    之前稱為"長期服務分支"，這是您已經熟悉的發布模式,每 2-3 年發布一個新的 Windows Server 主要版本。用戶有權享受 5 年的主流支持和 5 年的延長支持。

    授權¶

    當啟動一個帶有 Windows Server 基礎 AMI 的 Amazon EC2 實例時,Amazon 會為您承擔授權費用和授權合規性。
